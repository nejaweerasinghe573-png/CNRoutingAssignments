# CNRoutingAssignments
This project demonstrates the implementation of dynamic routing protocols in Cisco Packet Tracer. The network consists of multiple routers connected through WAN links and LAN networks. The objective is to configure, test, and compare RIP (Routing Information Protocol) and EIGRP (Enhanced Interior Gateway Routing Protocol).

Features

  Multi-router network topology
  RIP Version 2 configuration
  EIGRP configuration using Autonomous System (AS) 100
  Routing table verification
  End-to-end connectivity testing using ping
  Comparison of RIP and EIGRP performance
  
Technologies Used

  Cisco Packet Tracer
  Cisco IOS CLI
  RIP Version 2
  EIGRP
  
Learning Outcomes

  Configure dynamic routing protocols
  Verify routing information using Cisco commands
  Troubleshoot network connectivity issues
  Understand the differences between RIP and EIGRP
  
This project was completed as part of a networking laboratory assignment to gain practical experience in router configuration and dynamic routing.
